# Meteoromania Home Assistant Integration

## Overview
This Home Assistant custom integration fetches weather forecast data from Meteoromania for a specified location in Romania.

## Features
- Weather forecast for a specific location
- Sensors for:
  - Maximum temperature
  - Minimum temperature
  - Weather condition
- Hourly updates

## Installation
1. Add this repository to HACS
2. Install the Meteoromania integration
3. Configure the integration via the Home Assistant UI

## Configuration
1. Go to Configuration > Integrations
2. Click "+" to add a new integration
3. Search for "Meteoromania"
4. Enter your location (e.g., "IASI")
